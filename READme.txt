# Sales Dashboard & Analysis

## Overview
This is an interactive sales dashboard created using Power BI (ya Excel agar Excel mein banaya) to provide comprehensive insights into sales performance. The dashboard helps businesses track key metrics and make data-driven decisions.

## Features
- Total Revenue, Total Profit, and Quantity Analysis
- Average Price Calculation
- Monthly Sales Trend
- Top 5 Products by Sales
- Sales by Segment, Mode, Category & Sub-Category
- Key Performance Indicators (KPIs) including Previous Year Sales, Quantity, Average Sales, and Profit
- Sales Comparison by Year, Quarter, and Month

## Skills Demonstrated
- Dashboard Design & Visualization
- Analytical Skills & Business Insight Generation

## How to Use
- Open the Power BI (.pbix) file in Power BI Desktop 
- Interact with filters and visuals to explore the sales performance
